# Impeccable

Project design context for Sdone. Loaded by impeccable design skills.

## Design Context

### Users

A single owner-user (personal app, initially self-hosted). They keep Sdone open
to the side or in the background throughout a working day, dipping in to start a
work session, tick off a task, jot an interstitial note when interrupted, or
glance at the calendar. The primary context is a relaxed desktop window that
shares screen space with other work — so the interface must never demand
attention or add friction.

### Brand Personality

Three words: **calm, deliberate, personal.** Sdone ("it's done") should feel
like a well-kept paper notebook rather than a productivity dashboard shouting
metrics. Voice is quiet and encouraging, never gamified or urgent. Emotional
goal: a sense of steady, unhurried progress.

### Aesthetic Direction

- **Theme: light.** Daytime, background-open, journaling — light mode is the
  honest choice, not a safe default.
- Editorial/notebook feel: a warm literary serif for headings and reading
  content, a clean humanist sans for UI chrome.
- Restraint over decoration. Hairline borders, generous whitespace, one accent
  colour used sparingly (active nav, running session, primary action).
- Anti-references: gamified habit trackers, dense SaaS analytics dashboards,
  dark neon "focus" apps.

### Design Principles

1. **Low friction first.** The most common actions (start a session, add a
   task, log a note) are always one glance and one click away.
2. **Quiet by default.** Colour and emphasis are earned. The running session is
   the one thing allowed to stand out at any moment.
3. **Read like a notebook.** Typography and spacing prioritise comfortable
   reading; content is real, never placeholder.
4. **Calm motion.** Gentle, fast, ease-out transitions only. Always honour
   `prefers-reduced-motion`.
5. **Scales with features.** Nav must accommodate the full feature set (tasks,
   sessions, calendar, journal, achievements, stats, resources) without clutter.

### Type & Colour tokens (from spec 002)

- Headings / brand / reading content: **Literata** (warm literary serif).
- UI, labels, metadata: **Hanken Grotesk** (humanist sans).
- Colour moods explored (all light, OKLCH, neutrals tinted toward the accent):
  - Warm neutral + sage
  - Cool neutral + slate blue
  - Paper + terracotta
