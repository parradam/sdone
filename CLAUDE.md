# Sdone

Sdone (a contraction of "it's done") is a web application for task tracking.

## Purpose

Sdone will provide capabilities for:

- task tracking
- sessions per task and task type
- highlighting achievements (such as for performance reviews and CV building)
- interstitial journalling
- calendar view
- statistics
- link to resources or equipment required for tasks
